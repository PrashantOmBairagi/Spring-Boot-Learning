# Spring-SpringBoot 🌱

A hands-on learning repository for Spring Framework and Spring Boot,
covering core concepts through mini projects.

## 📁 Repository Structure

| Folder | Description |
|---|---|
| `src/main` | Core Spring - Dependency Injection, Java Config |
| `first-springBoot-project` | AOP - Aspect Oriented Programming |
| `Jdbc-Project-Minor` | Spring JDBC - Course Management Project |

## 🧠 Concepts Covered

- Dependency Injection
- Java Configuration Classes
- Aspect Oriented Programming (AOP)
- Spring JDBC
- Maven Project Setup

## 🛠️ Tech Stack

- **Language:** Java
- **Framework:** Spring, Spring Boot
- **Build Tool:** Maven
- **IDE:** IntelliJ IDEA

## 🚀 How to Run

```bash
# Clone the repo
git clone https://github.com/PrashantOmBairagi/Spring-SpringBoot.git

# Open in IntelliJ IDEA
# Run as Maven project
```
